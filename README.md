## Angular & .NET Project
**This Project is folling the Udemy tutorial "Learn to build an e-commerce app with .Net Core and Angular"**

**Required:** angular, dotnet, nuget, mssql, docker/rancher desktop

1. Clone
2. Open in IDE
3. Open docker/rancher
4. In terminal navigate to the project folder.
5. Run ```docker compose up -d```
6. Navigate to the API folder and run ```dotnet ef database update```
7. Run dotnet watch.
